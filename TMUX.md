# Commands
Your Tmux leader key is 'CTRL-B'.

From there you can type colon (:) to write out a command. Often we use: new-session [name]

- Detach: d
- Show all sessions: s
- Previous session: (
- Next session: )
- Create window: c
- Close window: &
- Previous window: p
- Next window: n
- Split pane vertically: %
- Split pane horizontally: "
- Switch to pane: Arrow keys
